info.kwarc.krextor

Class Krextor

  • java.lang.Object
    • info.kwarc.krextor.Krextor

  • public class Krextor
extends Object
    A Java wrapper for the Krextor XML→RDF extraction library, which is implemented in XSLT. Uses the Saxon XSLT processor. Part of this implementation is based on the at.srfg.ikewiki.importer.RXRReader class developed by Sebastian Schaffert for IkeWiki, which has been released under GPL.
    Author:
    Christoph Lange
